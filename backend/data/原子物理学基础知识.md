# 原子物理学基础知识

## 第一章 原子结构

### 1.1 原子的基本组成

原子是由原子核和核外电子组成的。原子核位于原子的中心，由质子和中子组成。质子带正电，中子不带电，电子带负电。

**关键概念：**
- 质子数决定了元素的种类
- 电子数等于质子数，使原子呈电中性
- 中子数可以不同，形成同位素

### 1.2 原子轨道

原子轨道是描述电子在原子核周围运动状态的数学函数。每个轨道可以容纳最多两个自旋相反的电子。

**轨道类型：**
1. **s轨道**：球形对称，每个能级有1个s轨道
2. **p轨道**：哑铃形，每个能级有3个p轨道
3. **d轨道**：复杂形状，每个能级有5个d轨道
4. **f轨道**：更复杂形状，每个能级有7个f轨道

**电子排布规则：**
- 泡利不相容原理：同一轨道最多容纳2个自旋相反的电子
- 洪特规则：电子优先单独占据能量相同的轨道
- 能量最低原理：电子优先填充能量较低的轨道

### 1.3 量子数

描述原子中电子状态需要四个量子数：

1. **主量子数 n**：决定电子的能量和轨道大小，n = 1, 2, 3, ...
2. **角量子数 l**：决定轨道形状，l = 0, 1, 2, ..., n-1
3. **磁量子数 m**：决定轨道在空间的取向，m = -l, ..., 0, ..., +l
4. **自旋量子数 s**：描述电子自旋，s = ±1/2

## 第二章 波粒二象性

### 2.1 光的波粒二象性

光既具有波动性，又具有粒子性，这种性质称为波粒二象性。

**波动性表现：**
- 干涉现象：杨氏双缝实验
- 衍射现象：单缝衍射、圆孔衍射
- 偏振现象：光的横波特性

**粒子性表现：**
- 光电效应：光子能量 E = hν
- 康普顿效应：光子具有动量 p = h/λ
- 光压：光子碰撞产生压力

### 2.2 物质波

德布罗意提出，不仅光具有波粒二象性，所有物质粒子都具有波动性。

**德布罗意关系式：**
- 波长：λ = h/p
- 频率：ν = E/h

其中 h 是普朗克常数，p 是动量，E 是能量。

**实验验证：**
- 戴维孙-革末实验：电子衍射
- 汤姆孙实验：电子通过多晶薄膜的衍射

### 2.3 不确定性原理

海森堡不确定性原理指出，粒子的位置和动量不能同时被精确测定。

**数学表达式：**
Δx · Δp ≥ ℏ/2

其中 Δx 是位置不确定度，Δp 是动量不确定度，ℏ = h/(2π)。

**物理意义：**
- 微观粒子不存在确定的轨道
- 只能用概率描述粒子的状态
- 测量会影响被测系统的状态

## 第三章 原子光谱

### 3.1 氢原子光谱

氢原子光谱是最简单的原子光谱，由一系列分立的谱线组成。

**光谱系列：**
1. **莱曼系**：紫外区，n₂ = 1
2. **巴尔末系**：可见光区，n₂ = 2
3. **帕邢系**：红外区，n₂ = 3
4. **布拉开系**：红外区，n₂ = 4
5. **普丰德系**：红外区，n₂ = 5

**里德伯公式：**
1/λ = R(1/n₁² - 1/n₂²)

其中 R 是里德伯常数，R = 1.097 × 10⁷ m⁻¹。

### 3.2 玻尔理论

玻尔提出了氢原子的量子化模型，成功解释了氢原子光谱。

**玻尔假设：**
1. **定态假设**：电子只能在特定轨道上运动，不辐射能量
2. **跃迁假设**：电子在不同能级间跃迁时，吸收或发射光子
3. **量子化条件**：角动量量子化，L = nℏ

**能级公式：**
Eₙ = -13.6/n² eV

**轨道半径：**
rₙ = n²a₀

其中 a₀ = 0.529 Å 是玻尔半径。

### 3.3 选择定则

原子在不同能级间跃迁时，必须遵守选择定则。

**电偶极跃迁选择定则：**
- Δl = ±1
- Δm = 0, ±1
- Δs = 0

不满足选择定则的跃迁称为禁戒跃迁，跃迁概率极小。

## 第四章 电子自旋

### 4.1 电子自旋的发现

斯特恩-盖拉赫实验发现，银原子束在非均匀磁场中分裂成两束，证明了电子具有内禀角动量——自旋。

**自旋特性：**
- 自旋量子数 s = 1/2
- 自旋磁量子数 mₛ = ±1/2
- 自旋角动量 S = √[s(s+1)]ℏ

### 4.2 自旋-轨道耦合

电子的自旋角动量和轨道角动量可以相互作用，产生自旋-轨道耦合。

**精细结构：**
- 能级分裂：原本简并的能级分裂成多个子能级
- 谱线分裂：单条谱线分裂成多条精细结构线
- 分裂间隔：与原子序数的四次方成正比

**总角动量：**
J = L + S

总角动量量子数 j = |l ± s|

### 4.3 泡利不相容原理

泡利不相容原理指出，在同一原子中，不能有两个电子处于完全相同的量子态。

**数学表述：**
两个电子的四个量子数 (n, l, m, mₛ) 不能完全相同。

**重要推论：**
- 每个轨道最多容纳2个电子
- 决定了元素周期表的结构
- 解释了原子的稳定性

## 第五章 多电子原子

### 5.1 中心力场近似

在多电子原子中，每个电子受到原子核和其他电子的共同作用。中心力场近似假设每个电子在一个等效的中心力场中运动。

**有效核电荷：**
Zₑff = Z - σ

其中 Z 是原子序数，σ 是屏蔽常数。

### 5.2 电子组态

电子组态描述了原子中所有电子的排布状态。

**常见组态：**
- 氢：1s¹
- 氦：1s²
- 锂：1s² 2s¹
- 碳：1s² 2s² 2p²
- 氧：1s² 2s² 2p⁴
- 氖：1s² 2s² 2p⁶

**洪特规则：**
在等价轨道中，电子优先以平行自旋方式单独占据轨道。

### 5.3 原子光谱项

原子光谱项用符号 ²ˢ⁺¹Lⱼ 表示，其中：
- S 是总自旋量子数
- L 是总轨道角动量量子数
- J 是总角动量量子数
- 2S+1 是多重度

**LS耦合：**
先将各电子的轨道角动量耦合成总轨道角动量 L，自旋角动量耦合成总自旋角动量 S，然后 L 和 S 耦合成总角动量 J。

## 第六章 原子的磁性

### 6.1 原子磁矩

电子的轨道运动和自旋都会产生磁矩。

**轨道磁矩：**
μₗ = -gₗμB√[l(l+1)]

**自旋磁矩：**
μₛ = -gₛμB√[s(s+1)]

其中 μB = eℏ/(2m) 是玻尔磁子，gₗ = 1 是轨道g因子，gₛ ≈ 2 是自旋g因子。

### 6.2 塞曼效应

原子在外磁场中，能级发生分裂，导致光谱线分裂，这种现象称为塞曼效应。

**正常塞曼效应：**
- 单重态原子（S = 0）
- 谱线分裂成3条
- 分裂间隔相等

**反常塞曼效应：**
- 多重态原子（S ≠ 0）
- 谱线分裂成多条
- 分裂间隔不等

### 6.3 斯塔克效应

原子在外电场中，能级发生分裂，导致光谱线分裂，这种现象称为斯塔克效应。

**线性斯塔克效应：**
能级分裂与电场强度成正比，主要出现在氢原子中。

**二次斯塔克效应：**
能级分裂与电场强度的平方成正比，出现在大多数原子中。

## 第七章 X射线

### 7.1 X射线的产生

X射线是高能电子轰击金属靶产生的高频电磁辐射。

**连续X射线：**
- 由电子的韧致辐射产生
- 具有连续的能量分布
- 最短波长由入射电子能量决定

**特征X射线：**
- 由内层电子跃迁产生
- 具有分立的能量
- 特征波长由靶材料决定

### 7.2 莫塞莱定律

特征X射线的频率与原子序数的关系：

√ν = a(Z - σ)

其中 a 和 σ 是常数，Z 是原子序数。

**物理意义：**
- 证明了原子序数的物理意义
- 确定了元素周期表的排列
- 发现了未知元素

### 7.3 X射线吸收

X射线通过物质时会被吸收，吸收系数随波长变化。

**吸收限：**
当X射线光子能量等于某一壳层电子的结合能时，吸收系数突然增大，这个波长称为吸收限。

**K吸收限、L吸收限、M吸收限**分别对应K、L、M壳层电子的电离能。

## 第八章 激光原理

### 8.1 受激辐射

爱因斯坦提出了三种辐射过程：

1. **自发辐射**：激发态原子自发跃迁到低能态，发射光子
2. **受激吸收**：原子吸收光子，从低能态跃迁到激发态
3. **受激辐射**：激发态原子在外来光子的激发下，跃迁到低能态，发射与入射光子完全相同的光子

### 8.2 粒子数反转

要实现激光，必须使高能级的粒子数大于低能级的粒子数，这种状态称为粒子数反转。

**实现方法：**
- 三能级系统
- 四能级系统
- 光泵浦、电泵浦等激励方式

### 8.3 激光特性

激光具有以下特性：

1. **单色性好**：频率范围极窄
2. **方向性好**：光束发散角极小
3. **相干性好**：时间相干性和空间相干性都很好
4. **亮度高**：单位面积单位立体角的功率极大

**应用领域：**
- 通信：光纤通信
- 医疗：激光手术、激光治疗
- 工业：激光切割、激光焊接
- 科研：光谱分析、精密测量
